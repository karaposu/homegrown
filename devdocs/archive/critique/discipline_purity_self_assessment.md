# Critique: Should Disciplines Evaluate Their Own Output?

## Verdict: Disciplines MEASURE. Inquiry ROUTES. Disciplines don't judge themselves.

### The user's point survives

Self-assessment makes disciplines impure — they do their cognitive operation AND evaluate whether it was sufficient. That's two operations in one discipline. And the rubber-stamping incident proved self-evaluation is unreliable.

### The distinction

| | Measurement | Judgment |
|---|---|---|
| **What it does** | Reports observable properties of the output | Decides whether the output is sufficient |
| **Example** | "5 perspectives, 3 produced new anchors, 80% ambiguities resolved" | "PROCEED — output is good enough" |
| **Who does it** | The discipline (it knows what it did) | Inquiry (it compares measurements to thresholds) |
| **Purity** | Pure — measurement is part of the cognitive operation | Impure — judgment is a second operation |

### How it works

```
Discipline:  cognitive work + REPORT MEASUREMENTS
             "perspectives: 5 (3 new anchors). ambiguities: 4/5 resolved. SV delta: significant."

Inquiry:     READ measurements + COMPARE to thresholds (from discipline spec) + ROUTE
             "perspective coverage ≥ 3? YES. ambiguity ratio ≥ 70%? YES. → PROCEED"
```

### What this means for the self-assessment sections

The measurement content stays. The "Overall: PROCEED/FLAG/RE-RUN" verdict line is reframed: it's inquiry's decision, not the discipline's. The discipline reports facts. Inquiry applies thresholds and routes.

### Three-way separation (clean)

- **Disciplines** → do cognitive work + report measurements (pure)
- **Inquiry** → compare measurements to thresholds + route (operational)
- **Critique** → adversarially evaluate candidates/ideas (its actual job)

No component does two jobs. Each stays in its lane.
