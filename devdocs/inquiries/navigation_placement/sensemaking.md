## SV6 — Stabilized Model

**Navigation is a BOUNDARY DISCIPLINE — independent, replaces wayfinding, operates between SIC iterations, paired with Reflection.**

### The Two-Tier Discipline System

```
PIPELINE DISCIPLINES (within one iteration):
  S → I → C
  Operate on: one question
  Produce: one answer
  
BOUNDARY DISCIPLINES (between iterations):
  R (Reflect): looks BACKWARD at the process
    Transform: SIC run → process observations + memory cells
  N (Navigate): looks FORWARD at the possibilities
    Transform: SIC output → navigation map of typed directions + guidelines

MVL: orchestrates both tiers
```

### Structural Finding

S, I, C share a property: they operate on the SAME question within ONE iteration. They're PIPELINE disciplines.

R and N share a different property: they operate on the RESULT of an iteration. They cross the iteration boundary. They're BOUNDARY disciplines.

N cannot be inside the pipeline (SICN) because its output (next questions) feeds the NEXT iteration, not the current one. The output's consumer is outside the producer's context — a structural mismatch.

### Answers

**1. Where N sits:** OUTSIDE the pipeline, at the iteration boundary. Paired with R.

**2. Does the primitive change?** NO. SIC stays. N is a separate discipline.

**3. MVL's role:** Orchestrates both tiers. At iteration-complete:
  - Lightweight fragments (frontier, gap, "narrow to X") for quick decisions
  - Suggests `/navigate` for full possibility map when strategic decisions needed
  - Suggests `/reflect` for process assessment when learning needed

**4. Own command?** YES. `/navigate` — independently useful, full discipline spec, required for autonomous loops.

### Navigation Replaces Wayfinding

Navigation is the SUPERSET. Wayfinding produced one direction. Navigation produces the full space. Wayfinding's "find highest-impact" becomes the SELECTION step after navigation's enumeration.

## Key Insights

- **I1**: S, I, C operate on the SAME QUESTION within ONE iteration (pipeline property). N operates on the RESULT and maps NEXT questions (boundary property). Structurally different.
- **I2**: R and N are symmetric boundary disciplines. R: backward (process). N: forward (possibilities).
- **I4**: N is independently valuable outside MVL — "what should I work on?" without SIC.
- **I5**: N doesn't map to alignment layers — it's a meta-operation ON the alignment cycle, not within it.
- **I7**: SICN has a structural mismatch: N's output consumer (next iteration) is outside the loop.
- **I8**: Human chooses depth: MVL fragments for quick, `/navigate` for strategic.
- **I10**: Navigation replaces wayfinding (superset — 15 types vs 6 moves).
- **I11**: R and N are paired meta-disciplines operating on the SIC cycle's output.

## Ambiguity Resolutions

1. SICN? → NO. N is outside the pipeline (boundary). Structural mismatch. **HIGH**
2. Own command or MVL logic? → Own command. Independently useful + explicit methodology + required for autonomy. **HIGH**
3. Replaces wayfinding? → YES. Navigation subsumes it. Wayfinding = navigation + implicit selection. **HIGH**

## How SV6 Differs from SV1

SV1: three options. SV6: boundary discipline, with two-tier system (pipeline vs boundary) as the structural finding.
