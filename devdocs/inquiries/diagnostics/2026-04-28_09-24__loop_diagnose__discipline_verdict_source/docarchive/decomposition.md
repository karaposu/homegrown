## Step 2 - Boundary Detection

### Boundary 1: Valid Result vs Failed Mechanism

The prior inquiry has a valid part and an invalid part:

- Valid: do not let disciplines authoritatively self-verdict.
- Failed: make routine Critique produce upstream marks.

This boundary prevents overdiagnosis.

### Boundary 2: Branch Framing vs Discipline Execution

The branch framing selected the design axis. The disciplines then worked mostly inside that axis.

This boundary matters because improving only Critique would not fix the root pattern if future branch questions still omit artifact-workflow alternatives.

### Boundary 3: Evidence Capture vs Diagnosis

The prior inquiry collapsed:

- recording evidence during the base loop;
- diagnosing failure from that evidence later.

This is the core conceptual boundary the corrected inquiry restored.

### Boundary 4: Routing vs Diagnosis

The prior inquiry let routability pressure shape diagnosis. The corrected inquiry separated routing convenience from diagnostic evidence.

This boundary explains why marks looked attractive in the prior loop.

### Boundary 5: User Proposal vs Design Candidate

The user's Critique-marks proposal should have been treated as one candidate. The prior loop treated it as a stronger anchor.

This boundary points to a maintenance candidate: expand user-proposed mechanisms into alternatives before selection.

## Final Decomposition

The diagnostic should evaluate six hypotheses:

1. **Loop framing failure:** the question centered verdict authority rather than diagnosis workflow.
2. **Sensemaking anchor dominance:** the user-proposed Critique-marks mechanism became too strong an anchor.
3. **Decomposition boundary error:** routine Critique evidence and retrospective diagnosis were grouped together.
4. **Innovation omission:** archive-first/on-demand diagnosis was not generated as a candidate.
5. **Critique dimension blindness:** base-loop weight, artifact reuse, and attribution scope were underweighted or absent.
6. **CONCLUDE overstatement:** the final finding made the surviving mechanism sound stronger than the evidence justified.
