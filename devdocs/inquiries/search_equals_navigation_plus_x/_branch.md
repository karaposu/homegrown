## L5: Prior Coherence
- Navigation enumerates WITHOUT selecting. It maps the space but doesn't choose.
- The old meta-search (`old_meta-search.md`) was about "designing and steering cognitive search processes." It was split into wayfinding (steering) + inquiry (loop management). 
- SIC IS a search — each cycle searches for an answer to a question.
- The full loop is: SIC → R → N → Select → next SIC → ... → goal reached.
- Selection (choosing from N's map) is the part between navigation and the next cycle.
- The GOAL (from _branch.md) is the termination condition — search ends when the goal is met.
