# Branch: Preparation Step

## Question
Should there be an optional preparation step before the SIC loop that establishes current and future scope, strategic context, and possibly other benefits — and if so, what does it produce, how does it persist, and what value does it add beyond scope checking?

## Goal
The user should know: (1) whether a preparation step is worth adding, (2) what it would produce concretely, (3) what benefits it provides beyond scope (adapter selection? context loading? goal refinement?), (4) how its output persists across inquiries, and (5) whether it should be mandatory or optional.

## Scope Check
Question covers goal: YES — the question asks about scope, persistence, additional benefits, and optional/mandatory, which matches all five goal points.
