# Branch: SIC as Wayfinder

## Question
How can the SIC loop (MVL) be used to perform wayfinding — steering toward the most impactful next action and navigating toward a goal — given that wayfinding was identified as a structurally distinct cognitive shape from S, I, and C?

## Goal
A practical answer the user can apply immediately: either a concrete method for using `/MVL` to do wayfinding work, or a clear explanation of why SIC can't fully subsume wayfinding and what the minimal complement is. The user should walk away knowing exactly what to do when they need to steer.
