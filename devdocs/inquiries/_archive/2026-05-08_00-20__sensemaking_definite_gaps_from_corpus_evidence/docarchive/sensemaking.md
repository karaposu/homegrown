## Phase 3 — Ambiguity Collapse

### Ambiguity 1: Should Rule A's Phase 5 sub-aspects (proxy-vs-structural, discoverability, user-language alignment) be EXPLICITLY enumerated in the rule wording, or kept as ILLUSTRATIVE examples?

**Strongest counter-interpretation:** enumerate explicitly. Each sub-aspect was derived from a separate chain's primary-cause finding (S1, T1, U1). Burying them as "illustrative examples" loses the evidence trail.

**Why the counter-interpretation fails (structural grounds):** the rule should be GENERIC. Future chains may surface a 4th, 5th, etc. Phase 5 sub-aspect. Locking the rule to exactly three named sub-aspects creates maintenance burden (every new sub-aspect requires updating the rule's enumeration). Illustrative examples accept future additions naturally. Plus: the corpus's M6-refinement-S2 wording (chain 6 PROMOTION) covers Phase 5 broadly; T1 + U1 are sister rules under the broader umbrella, not co-equal enumerated members. The natural reading is "Phase 5 layer + sub-aspects accumulate over time."

**Confidence:** HIGH (illustrative examples preserve generic phrasing + future extensibility).

**Resolution:** Phase 5 sub-aspects are stated as illustrative examples in Rule A's wording, with explicit citation to the chain findings (S1, T1, U1) as the evidence base. Future sub-aspects can be added as more chains accumulate; the rule's general shape stays stable.

**What is now fixed?** Rule A's wording uses illustrative examples for Phase 5 sub-aspects, not exhaustive enumeration.

**What is no longer allowed?** Locking the rule to exactly three named Phase 5 sub-aspects.

**What now depends on this choice?** Innovation generates Rule A's wording with examples-not-enumeration shape.

If discoverability for Phase 1 / SV2+ / Phase 5 readers is a concern, optional one-line cross-references at those layers can point to Phase 3 (per the convention's optional-cross-reference pattern). But the canonical home is single: Phase 3.

**Confidence:** HIGH (placement convention is unambiguous on step-level scope).

**Resolution:** Rule A's canonical home is Execute section's Phase 3 — Ambiguity Collapse. Optional one-line cross-references at Phase 1 / Phase 2 / Phase 5 are NOT added by default (per the user's purity preference); they can be added later if discoverability proves insufficient. **What is now fixed?** Rule A's canonical home is Phase 3. Mandatory cross-reference at Premature Stabilization. Optional cross-references at Phase 1 / Phase 2 / Phase 5 are NOT added at this run.
