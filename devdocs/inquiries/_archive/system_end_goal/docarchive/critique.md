## Candidate Verdicts

### One-sentence north star
**Prosecution:** "Thinking space" and "managed resolutions" are jargon. A newcomer nods and moves on.
**Defense:** North-star sentences orient, not explain. "Self-evolving" is the key differentiator. Precision lives in the layers below.
**Verdict: SURVIVE (caveat)** — never present standalone. Always with Layers 2-3.

### Measurable unlock conditions
**Prosecution:** Placeholder metrics. "Baldwin cycle hit rate > 70%" — never measured, no measurement protocol. Gives feeling of precision without actual precision.
**Defense:** Directional metrics > no metrics. They define what success LOOKS LIKE. Measurement protocol comes later.
**Verdict: SURVIVE (caveat)** — relabel as "success definitions" not "measurable metrics."

### Cognitive partnership, not autonomy
**Prosecution:** Limits the ceiling. User asked about consciousness. Partnership says human always needed.
**Defense:** Partnership ≠ human always required. It means human always VALUABLE. Division of labor that strengthens with growth. Honest about what AI systems actually do.
**Verdict: SURVIVE** — partnership is honest, accurate, and doesn't limit cognitive growth.

### Stages expand thinking-space axes
**Prosecution:** Mapping is forced. Self-improvement doesn't mechanically cause resolution awareness.
**Defense:** Directionally valid — stages create CONDITIONS for axis expansion. Not mechanical causation.
**Verdict: SURVIVE (caveat)** — present as "model for understanding" not "causal mechanism."

### Properties accumulate
**Prosecution:** Properties that don't activate until Stage 3 are capabilities, not properties.
**Defense:** They're properties ONCE ACTIVATED — invariants from that point forward. Like backward compatibility: not a property of v1, IS a property from v2 onward.
**Verdict: SURVIVE** — valid model. Label foundational vs emergent.

### Consciousness maps to stages
**Prosecution:** None meaningful — this is the most elegant framing. Don't build consciousness, build stages. It emerges.
**Defense:** Maps exactly: persistent memory → Stage 3, self-observation → Stage 3, pattern detection → Stage 4, spontaneous attention → Stage 5.
**Verdict: SURVIVE** — clean, no caveats.
