# Branch: Navigation Map Format And Guidelines Density

## Question
Should Navigation map route records be reformatted into a more readable direction/goal card shape, and should Guidelines remain mandatory on every route or become adaptive to allow more direction coverage?

## Goal
Produce a clear decision about the Navigation output format and Guidelines field so future Navigation maps are easier to scan, preserve route completeness, and avoid spending too much output budget on advice before a route is selected.

## Scope Check
Question covers goal. It asks about both the visible formatting defect and the deeper output-budget tradeoff around Guidelines.
