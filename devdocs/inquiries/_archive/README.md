# Inquiry Archive

Cold inquiry folders live here. This archive preserves provenance; it is not deletion.

Use this folder for completed, superseded, bad, abandoned, or stale inquiry folders that no longer belong in the active `devdocs/inquiries/` surface.

Do not move active or likely-to-resume inquiries here without an explicit decision.

Reference policy:

- update active/resumable references;
- update recent or important finding references when practical;
- tolerate stale references inside old archived provenance when the rewrite cost is not worth it.

Preferred new inquiry naming convention after runner adoption:

```text
YYYY-MM-DD_slug
```
