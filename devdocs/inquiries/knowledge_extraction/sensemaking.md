### How It Works

```
User has: folder with many md files, some contradicting
User runs: /sense-making folder/
S reads ALL files → extracts anchors → resolves contradictions → SV6 = pure version
```

S operates at the ANCHOR level (individual insights), not the FILE level. Files are containers. S extracts the atoms (anchors) and rebuilds the molecule (SV6) from surviving atoms, regardless of which file they came from.

### Resolution Method

S resolves contradictions by EVIDENCE STRENGTH, not recency. The strongest argument wins. More recent isn't always more correct. If recency matters, the user provides that context.
