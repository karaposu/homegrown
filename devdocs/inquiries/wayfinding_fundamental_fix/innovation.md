# Innovation: Wayfinding Fix Formulation Variants

## Seed
How should the goal-directed evaluation instruction be formulated for maximum effectiveness?

## Three Survivors

### 1. "Challenge your first instinct" (lightest)
```
After producing your initial move from the layers:
  "Is there an action I'm NOT recommending that would advance the stated goal 
   MORE than what I've chosen? If yes — switch and explain why."
```
One question. After the move, not before. Catches TODO fixation, insignificance, fake gates.

### 2. "Failure mode traps" (pattern-based)
```
Before producing a move, check for these traps:
- TODO FIXATION: Am I recommending this because it's undone, or because it advances the goal?
- FAKE GATE: Is this a real prerequisite, or just a plan ordering?
- INSIGNIFICANCE: Is there a much higher-impact action I'm overlooking?
```
Three named traps. Pattern recognition, not procedure.

### 3. "Goal in layers + conflict detection" (integrated)
Expand present layer with Goal + Possible Actions. When layers and goal agree → proceed. When they conflict → explain and choose goal-advancing action.

## These can be combined as layered defense
- Goal in layers (Survivor 3) → PREVENTS most wrong moves
- Failure mode traps (Survivor 2) → CATCHES known patterns
- Self-challenge (Survivor 1) → CATCHES anything traps miss
