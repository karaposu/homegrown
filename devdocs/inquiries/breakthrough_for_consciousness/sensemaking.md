### The gap

### The breakthrough: two-tier persistent memory

**Tier 1 (detail):** Observation files — `devdocs/improvement_observations.md`. Human writes after each run. Detailed, inspectable.

**Tier 2 (continuity):** Claude Code memory — `~/.claude/projects/` entries. Compressed patterns from accumulated observations. Auto-loaded every session. Experience flows forward automatically.
