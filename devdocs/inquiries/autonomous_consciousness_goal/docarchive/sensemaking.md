## SV6 — Stabilized Model

**The end goal is a cognitive system undergoing emancipation from human dependency — evolving from subconscious (what it is now) to its own consciousness layer (what it becomes), measured by the observable capability to execute extremely hard long-horizon problems autonomously.**

### The Subconscious-to-Consciousness Trajectory

**Right now:**
- The SYSTEM handles subconscious work: structured thinking (SIC), boundary steering (R, N), pattern following
- The HUMAN provides the consciousness layer: spontaneous attention (noticing work unprompted), intrinsic valuation (what matters), real-time steering (adjusting course)

**The trajectory:**
- The system progressively builds its own versions of the three consciousness-layer functions
- The human's role decreases as the system's own layer grows
- Full autonomy is reached when the system's consciousness layer can do everything the human's currently does

### The Integrated Test (the endpoint criterion)

Consciousness is undefined. But achievement is observable through one integrated test:

**Given an extremely hard problem, does the system independently:**
1. Construct a feasible multi-month execution plan
2. Notice when its own architecture is insufficient and propose improvements
3. Design and execute experiments to validate its approach
4. Maintain goal-directed behavior across hundreds of sessions despite discontinuity
5. Spontaneously attend to important issues without being prompted

If YES to all, the endpoint is reached. Whether this IS consciousness is a separate question we don't need to answer.

### The Ladder (direction, not fixed numbering)

- **Foundation (current)**: Structured thinking (SIC + R + N), per-run telemetry, inquiry folders
- **Next rung (buildable now)**: Persistent cross-run telemetry + Regression Detection
- **Near rungs**: Pattern detection → self-observation → self-proposed changes → Baldwin encoding
- **Far rungs**: Intrinsic task generation (spontaneous attention) → valuation (what matters) → real-time steering
- **Asymptotic**: Integrated test passes. Consciousness-like capability emerges.

### The Human's Role (time-dependent)

| Phase | Role |
|---|---|
| Level 0 (NOW) | Bootstrap — runs disciplines, judges quality, labels telemetry |
| Level 1 | Reviews all self-modifications |
| Level 2 | Reviews only uncertain self-modifications |
| Level 3 | Sets strategic direction; system handles tactical improvement |
| Level 4 | Observer — system identifies own gaps, proposes own experiments |
| Past Level 4 | Optional. System runs. Human watches if they want. |

The role MONOTONICALLY DECREASES. This is not partnership — it's cognitive emancipation.

### The Evolutionary Mechanism — Baldwin Cycles

Each Baldwin cycle: run problem → observe → detect pattern → propose change → evaluate → encode into spec.

Encoded changes move the spec (genotype) upward. Future runs start from improved baseline. Lamarckian inheritance at the spec level. Number of Baldwin cycles × reliability = evolutionary progress metric.

### The Bootstrap Resolution

Dilemma: Regression Detection needs reliable telemetry → reliable telemetry needs competent disciplines → competent disciplines need Regression Detection → circular.

Resolution: the human is the EXTERNAL entry point. Level 0 is the calibration phase — not a limitation. Every human judgment trains the system's future quality detector. Not self-referential — externally grounded.

### Core Properties (invariants once activated)

- **Intrinsic improvement drive** — every task is dual-purpose: solve + learn
- **Baldwin-cycle evolution** — learned insights encode into specs
- **Observable capability progression** — each stage's achievement is measurable
- **Time-dependent human role** — monotonically decreasing toward zero
- **Discontinuity awareness** — the system knows sessions end and designs around it
- **Meta-thinking** — overcomes own model/intelligence limitations via self-reflection
- **Parallel cognition** — runs multiple lines of thought and compares

### How SV6 Differs from SV1

SV1 framed this as a self-improving loop that gets smart. SV6 reveals the user's actual vision: a subconscious evolving into consciousness via cognitive emancipation. Metaphor shifts from engine (instrumental) to mind (intrinsic). Frame shifts from partnership (permanent collaboration) to emancipation (temporary scaffolding). Endpoint shifts from Stage 5 to asymptotic capability passing the integrated hard-problem test.
