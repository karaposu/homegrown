# Sensemaking: Is Regression Detection a Protocol?

## SV6 — No. It's system-level failure mode awareness.

### Not a protocol
Protocols route — they don't judge. Regression detection is entirely about judging quality decline. The protocol definition explicitly excludes quality evaluation.

### Not a discipline
Disciplines transform cognitive states (ambiguity → understanding, opaque → model). Regression detection monitors without transforming. No cognitive transform, no core operation.

### What it IS
System-level failure mode awareness — the same concept as the failure modes inside each discipline, applied at the system scope.

```
Discipline failure modes = "how THIS DISCIPLINE fails"
  (Status Quo Bias, Surface Fluency, Paradigm Projection, etc.)

Regression symptoms = "how THE WHOLE SYSTEM declines"
  (Surface Run, Confirmation Bias, Introduced Error, Slow Drift, etc.)

Same concept. Different scope.
```

### Where it lives

```
thinking_disciplines/
  ├── [discipline specs]     → cognitive operations
  ├── protocols/             → operational procedures
  └── regression/            → system-level failure mode awareness
```

Three peer components. Not inside protocols. Not a discipline. A third structural element.
